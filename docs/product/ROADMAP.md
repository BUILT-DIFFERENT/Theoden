# Roadmap

## Phase 1

- Shell app with theming, control room, and thread view.
- Read shared CLI config and history via `codex app-server`.
- UI shows threads and run timeline.

## Phase 2

- Start local runs via CLI.
- Stream events into the timeline.
- Support multiple concurrent runs.

## Phase 3

- Diff viewer and review panel.
- One-click commit with auto-generated message.
- Commit and push, or commit and create PR.

## Phase 4

- Worktree creation and lifecycle.
- Bring-back merge assistant.
- Visual mapping between thread, worktree, and branch.

## Phase 5

- Automations scheduler.
- Skills browser with Yeet and initial integrations.

## Phase 6

- Cloud provider UI complete.
- Provider interface ready for real cloud execution.
