# Integrations

## External editors

- Detect VS Code and JetBrains family via registry and common install paths.
- Provide Open in editor and Open changed file actions.
- Never block the user from switching tools.

## Terminal

- Provide run-command panel for a simple implementation.
- Provide Open terminal here for Windows Terminal.
- Optional embedded terminal panel with streaming output.

## Clipboard and attachments

- Support paste and drag-drop images.
- Store attachments on the thread.
- Pass attachments to the CLI when supported.
